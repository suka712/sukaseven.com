## hey, i'm khiem

full-stack engineer based in taiwan. i build identity systems, real-time infrastructure, and the occasional dev tool that scratches my own itch.

this isn't a resume — it's a war room. everything you see is wired to real data: commits, service health, what i'm currently listening to.

### what's shipping

- **katanaid** — auth platform built on go + postgres. oauth2, sessions, the whole thing.
- **caphne** — real-time collaboration. websockets, operational transforms, shared cursors.
- **anyu** — s3-compatible file service. presigned urls, multipart, the works.
- **hasaki-go** — cli tool that writes your git commit messages using gemini.

### how to navigate

use the file tree on the left — each entry opens a devlog in this panel. or use the terminal at the bottom. try `ls projects` or `cat projects/Caphne` to start.

keyboard shortcuts are in the bottom-right corner.

### stack, briefly

**frontend:** next.js, typescript, tailwind
**backend:** go, postgres, redis, docker
**infra:** railway, cloudflare, github actions
